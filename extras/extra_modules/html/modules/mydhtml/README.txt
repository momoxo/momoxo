mydhtml
=======
SmartMarkUP (BBcode Editor for textarea) for KARIMOJI
This module is a just sample how to implement BBCode editor as KARIMOJI module.

Requirement
-----------
-KARIMOJI 2.2 or later

Main Feature
------------
- Implement SmartMarkUP(http://www.phpcow.com/smartmarkup) to textarea

How to Use
----------
1) Install this module as normal.
2) Change the target text area's "editor" parameter as <{xoops_dhtmltarea editor=bbcode}>
